<h1>🎸 Responsive Guitar Website</h1>

<p>A modern and responsive landing page for a fictional guitar brand called <strong>Guitars World</strong>.  
This project focuses on building a clean user interface using <strong>HTML5</strong> and <strong>SCSS</strong>, while applying modern layout techniques such as <strong>Flexbox</strong> and <strong>CSS Grid</strong>.</p>


<h2>🛠 Technologies Used</h2>
<p>
<ul>
  <li>HTML5</li>
  <li>SCSS (Sass)</li>
  <li>CSS3</li>
  <li>Flexbox</li>
  <li>CSS Grid</li>
  <li>Font Awesome</li>
</ul>
</p>

<h2>✨ Features</h2>
<p>
<ul>
  <li>Responsive layout for different screen sizes</li>
  <li>Clean and modern landing page design</li>
  <li>Feature cards section</li>
  <li>Product image gallery</li>
  <li>Fullscreen navigation menu</li>
  <li>Organized SCSS structure</li>
  <li>Mobile-friendly interface</li>
</ul>
</p>


<h2>🎯 Project Purpose</h2>
<p>The goal of this project was to practice:</p>
<ul>
  <li>Structuring a responsive webpage</li>
  <li>Writing modular and maintainable styles using SCSS</li>
  <li>Creating reusable UI components</li>
  <li>Improving front-end development skills</li>
</ul>

<h2>📱 Responsive Design</h2>
<p>The layout adapts to multiple devices:</p>
<ul>
  <li>Desktop</li>
  <li>Tablet</li>
  <li>Mobile</li>
</ul>
